# Documentation: Posts, Pages, Media

#  Task 1: Implement Core Concepts (Posts, Pages, Media)

##  Posts
Posts are used for dynamic content like blogs & in wordpress it is like title and paragraph.

### Steps:
1. Go to Dashboard → Posts → Add New
2. Enter title and content.
3. Add categories and tags, example :-category :technologty
   tags: mobile, ipad, laptop, headset.
5. Click **Publish**
6. it will create new page with categories and tags.

---

##  Pages
Pages are static content like About, Contact,our services,skills & more.

### Steps:
1. Go to Dashboard → Pages → Add New
2. Enter title and content
3. Click **Publish**
4. It will create new page.

---

##  Media
Using Media we can include images, videos, and files.

### Steps:
1. Go to Dashboard → Media → Add New
2. Upload files (images/videos)
3. Use in posts/pages

---

#  Task 2: Create / Configure Features

## Posts Configuration
- Create Categories: Posts → Categories
- Create Tags: Posts → Tags
- Assign category/tag while creating post

##  Pages Configuration
- Create important pages:
  - Home
  - About
  - Contact
- Set Homepage:
  - Settings → Reading → Static Page

##  Media Configuration
- Upload optimized images
- Add ALT text for SEO
- Organize media library

---

#  Task 3: Customize UI / Settings

## Posts UI
- Use block editor (Gutenberg)
- Add blocks:
  - Paragraph
  - Image
  - Heading

##  Pages UI
- Edit using Elementor (optional)
- Customize layout and sections

##  Media UI
- Edit image details:
  - Title
  - Caption
  - ALT text

---

#  Task 4: Debug / Optimize

## Common Issues & Fixes

### Images not loading
- Check media upload
- Re-upload image

###  Slow website
- Compress images
- Use caching plugin

###  Post not showing
- Check publish status
- Verify category settings
